# IINTEKKU Group Website

A modern, professional corporate website for IINTEKKU Group built with Next.js, TypeScript, and Tailwind CSS.

## Brand Philosophy

**Execution. Efficiency. Partnership. Results.**

**Together, We Shall Thrive.**

## Project Overview

The IINTEKKU Group website represents a growing organization built around business solutions, operational excellence, long-term partnerships, and future expansion. The site is designed to be clean, modern, responsive, and comparable in quality to leading global consulting and business services firms.

## Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Forms:** React Hook Form + Zod
- **Package Manager:** npm

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/roubensbacette-crypto/IIGC.git
cd IIGC

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
- `npm run type-check` - Run TypeScript type checking

## Project Structure

```
src/
├── app/                  # Next.js app directory
├── components/          # Reusable React components
│   ├── layout/         # Layout components (Header, Footer, Navigation)
│   ├── sections/       # Page sections
│   ├── ui/             # Atomic UI components (Button, Card, Input, etc.)
│   └── forms/          # Form components
├── pages/              # Next.js pages
├── styles/             # Global styles
├── types/              # TypeScript type definitions
├── lib/                # Utility functions and helpers
└── utils/              # Helper functions
```
